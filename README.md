## What's inside

  The 5 Pillars → JARVIS proof-of-capability → Journey → Join. Zero dependencies, pure
  HTML/CSS/JS, starfield + scroll animations, "Day X" live counter, integrity footer.

  proposals → discuss → vote YES/NO/**VETO** (≥33% veto blocks) → treasury with auto fund
